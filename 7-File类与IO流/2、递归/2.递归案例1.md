# 递归案例：和与阶乘

`题目1`

​		使用递归方式，计算1~n的和。

```java
public class Demo01 {
    public static void main(String[] args) {
        int result = sum(10);
        System.out.println(result);//55
    }

    private static int sum(int n) {
        if(n == 1){
            return 1;
        }
        return n + sum(n-1);
    }
}
```



`题目2`

​		使用递归的方式计算n!。

```java
public class Demo01 {
    public static void main(String[] args) {
        int result = add(5);
        System.out.println(result);//120
    }

    private static int add(int n) {
        if(n == 1){
            return 1;
        }
        return n * add(n-1);
    }
}
```

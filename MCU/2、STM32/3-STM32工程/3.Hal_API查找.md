## 1、<span style="color:brown">本地文档：</span>

STM32CubeMX帮助菜单 → 查看HAL文档路径 → 打开帮助文档：

<img src="https://raw.githubusercontent.com/root-bine/image/main/Typora-image/Hal_API_Search01.png" alt="image-20251111193710285" style="zoom:40%;" />

根据路径找到Repository文件夹，根据**芯片的型号**选择`STM32Cube_FW_Fx_Vx.x.x`，找到以下路径：

<img src="https://raw.githubusercontent.com/root-bine/image/main/Typora-image/Hal_API_Search02.png" alt="image-20251111194309921" style="zoom:40%;" />

打开其中任一文件：

<img src="https://raw.githubusercontent.com/root-bine/image/main/Typora-image/Hal_API_Search03.png" alt="image-20251111194539942" style="zoom:67%;" />

根据**需要使用的库**，选择**HAL库** `or` **LL库**。

**1.1、接线：**

通过**USB线**连接**开发板**和**PC**，若<u>电脑**没有安装过USB转串口驱动**</u>：

- 在设备管理器中，会**没有COM口**，并**提示驱动提示叹号**；

下载并安装`ch341ser.exe`，待安装成功后，连接`PC`与`MCU`。在`PC`的**设备管理器 -> 端口**显示：

## 2、<span style="color:brown">烧录配置：</span>

**2.1、LED灯闪烁：**

```C
#include <reg52.h>
sbit led = P2 ^ 7; // LED1
//带参延时函数
void delay_ms(unsigned int xms) {
    unsigned int i, j;
    for(i=xms;i>0;i--) {
        for(j=124;j>0;j--) {}
    }
}
void main() {
	led = 0;
    delay_ms(500);
    led = 1;
    delay_ms(500);
}
```

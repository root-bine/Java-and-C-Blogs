## 2、<span style="color:brown">debug操作：</span>

**2.1、debug操作：**

<img src="https://raw.githubusercontent.com/root-bine/image/main/Typora-image/MDK_DeBug03.png" alt="image-20251116175128860" style="zoom:80%;" />

|                   分类                   |                   说明                    |
| :--------------------------------------: | :---------------------------------------: |
|    `Disassembly Windows`（汇编窗口）     |               查看汇编代码☑️               |
| `Call Stack Windows`（堆栈局部变量窗口） |              查看各种变量值☑️              |
|       `Watch Windows`（观察窗口）        |               查看全局变量☑️               |
|     `Serial Windows`（串口打印窗口）     |         显示从串口打印出来的内容✔️         |
|     `Memory Windows`（内存查看窗口）     | 根据输入地址，查看**该片内存**的变化情况☑️ |
|     `Trace Windows`（性能分析窗口）      |   查看**各个函数执行时间和**所占百分比☑️   |
| `System Analyzer Window`（逻辑分析窗口） |           观察`IO`口的电平变化            |

**2.2、示例：**

在代码中，添加全局变量a、b、c，在`main`函数中进行初始化，并在逻辑代码中进行数据更新：

````c
// ================= 全局变量 =================
unsigned int a, b, c;
// ================= 局部变量 =================
a = b = c = 10;
// ================= while(1) =================
if(HAL_GPIO_ReadPin (GPIOC, GPIO_PIN_0 ) == 0) {
	HAL_Delay(10);
    if(HAL_GPIO_ReadPin (GPIOC, GPIO_PIN_0 ) == 0){
        HAL_GPIO_TogglePin (GPIOA, GPIO_PIN_6);
        a++;
		b--;
        c = a + b;
		while(HAL_GPIO_ReadPin (GPIOC, GPIO_PIN_0 ) == 0);
    }
}
````

编译并烧录后，开启debug模式：

- 添加`a`、`b`、`c`到`Watch Windows`：选中变量，鼠标右键点击`add 'x' to`；
- 先进行`ReSet`，再点击`Run`，此时点击触控按键，观察变量数值变化；

# Java基础了解

## Java三大版本（Write Once,and Run Anywhere):

JavaSE：标准版（桌面程序、控制台开发......）

JavaME：嵌入式开发（手机、小家电......）

JavaEE：E企业级开发（web端、服务器开发......）

## JDK \ JRE \ JVM:

JDK:Java Development Kit

JRE:Java Runtime Environment

JVM:Java Virtual Machine//简称：Java虚拟机（跨平台实现基础）

## JDK卸载：

删除JDK安装目录

删除Java_Home

删除path下关于Java的目录

java -version

## JDK安装：

1. 找到 JDK8，下载
2. 根据版本选择JDK版本
3. 安装JDK
4. 记住安装路径
5. 配置环境变量

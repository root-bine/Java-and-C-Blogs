## 1、<span style="color:brown">Timer概述：</span>

**1.1、简介：**

定时器，是**用来控制时间**，可以根据**设定的时间间隔**，准时产生**触发信号**，用于**控制程序的执行流程**。

`51`单片机有两个定时器：`Timer0`、`Timer1`。

**1.2、问题引入：**

之前使用**软件空转延时方式**（`void delay_ms(unsigned int xms)`），让`CPU`通过**不停的计数来消耗时间**。

但该方式有弊端，就是<span style="color:red">当`CPU`空转延时的时候，无法做其他事情</span>。

**2.2、工作模式分类：**

`STC89C52RC/RD+`的`Timer0`、`Timer1`有**4种工作模式**：

| Mode_Name                                         |        Explanation         |
| :------------------------------------------------ | :------------------------: |
| Mode_0                                            |      **13位**`Timer`       |
| **Mode_1**（<span style="color:red">常用</span>） |      **16位**`Timer`       |
| Mode_2                                            | **8位自动重装模式**`Timer` |
| Mode_3                                            |       两个8位`Timer`       |

除开**Mode_2**，其他模式均需要<span style="color:red">在**触发溢出中断**后，**重新配置**预装载值</span>。

## 1、<span style="color:brown">GPIO电平：</span>

**1.1、点亮本质：**

**`LED`点亮**，是通过控制`GPIO`**输出高低电平**：

- `GPIO = LOW_LEVEL -> LED_ON`；
- `GPIO = HIGH_LEVEL -> LED_OFF`；

**1.2、51单片机IC手册：**

`STC89C52RC / STC89LE52RC`芯片手册，每组`GPIO`的**电平状态**都被储存在**一个单独的`8 bit_Register`**：

## 2、<span style="color:brown">操作Register：</span>

**2.1、原理图：**

`STC89C52RC / STC89LE52RC`中，`LED`接入`P2 GPIO_Port`：`P2.7 -> LED1`，`P2.6 -> LED2`，`P2.5 -> LED3`。

<img src="https://raw.githubusercontent.com/root-bine/image/main/Typora-image/LED07.png" alt="image-20250909163125475" style="zoom:50%;" />

**2.2、操作整个GPIO：**

> `#include <reg52.h>`中，记录`STC89C52RC/STC89LE52RC`芯片的`Register Message` 
>
> `53H`、  `01010011B`，为**嵌入式单片机语法**，区分**C语法** 

点亮`LED`，可以**直接操作<u>整个GPIO</u>**，即：

```Java
#include <reg52.h>
void main(){ // LED1、LED3亮, LED2熄灭
  P2 = 0x53;  // 0101 0011
}
```

**2.3、控制Register位：**

`sbit`，是**C51编译器的拓展关键字**，用于<u>**定义**`Register`的**位变量**</u>：

```c
sbit led1= P2^7;
led1= 0;
```

```C
#include <reg52.h>
sbit led = P2^7;
void main() { // LED1亮
    led = 0;
}
```

`Buzzer`连接`MCU`的`GPIO P2`口：`P2.1`。



## 2、<span style="color:brown">代码实现：</span>

```C
#include <reg52.h>
sbit speak = P2^1;
void delay_ms(unsigned int xms) {
    unsigned int i, j;
    for(i = xms; i > 0; i--)
        for(j = 124; j > 0; j--);
}
void beep(unsigned int time_ms) {
    unsigned int t;
    for(t = 0; t < time_ms; t++) {
        speak = 1;
        delay_ms(1);
        speak = 0;
        delay_ms(1);
    }
}
void main() {
    while(1) {
        beep(200);   // 发声约400ms（即“滴——”）
        delay_ms(500); // 停500ms
    }
}
```
